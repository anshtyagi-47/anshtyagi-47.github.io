Today I started learning React through Code with Mosh's React course. The first lesson introduced the basics of React and explained why it has become one of the most popular libraries for building modern web applications.

I learned how React uses a component-based architecture, where the user interface is divided into small, reusable pieces called components. This approach makes applications easier to organize, maintain, and scale. I also understood how React updates only the parts of the page that change by using the Virtual DOM, which improves performance.

After understanding the fundamentals, I set up the React development environment and created my first React project. I explored the project structure and learned the purpose of important files such as `main.jsx`, `App.jsx`, and the `src` folder.

### What I Learned

- What React is and why it's used
- Benefits of component-based development
- Virtual DOM and efficient rendering
- Setting up a React development environment
- Understanding the basic project structure
- Running a React application locally

Starting React feels like an exciting step in my web development journey. I'm looking forward to learning JSX, components, and building interactive user interfaces over the next few days.
